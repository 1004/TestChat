一、简介
该款APP是一个后台基于bmob后端云的校园社交APP，后台采用bmob云存储技术。界面采用了谷歌的matrial design设计，框架基于MD+Rxjava+retrofit+MVP架构。
到目前为止，已经完成的功能模块有单聊，群聊，附近人搜索，开心时刻，天气预报，朋友圈发表和个人信息编辑展示等7大功能模块。
